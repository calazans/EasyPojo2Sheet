### Por que EasyPojo2Sheet?

- **Zero dependências externas** (nem mesmo Lombok)
- 100% type-safe com anotações em tempo de compilação
- Suporte nativo a estilos, freeze pane, auto-size
- Funciona perfeitamente com Spring Boot, Quarkus, Micronaut e Jakarta EE
